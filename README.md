# Length Converter
A program that converts a length in feet and inches to a length in meters and centimeters.

## Setup
1. Download and install [Git](https://git-scm.com/downloads), if you don't already have it.

2. Open terminal and run the git clone command:

   ```
   $ git clone https://github.com/cramaechi/length-converter.git
   ```
    or [download as ZIP](https://github.com/cramaechi/length-converter/archive/master.zip).

3. Go to the project folder:

   ```
   $ cd length-converter
   ```

4. Compile:

   ```
   $ make
   ```
   
## Usage
Run:

```
$ ./length_converter
```
